# 🎮 Hot Seat Challenge

Hot Seat Challenge is a console-based quiz game developed in **C language**, inspired by the popular quiz show 🎤 Kaun Banega Crorepati (KBC).

The game allows players to test their knowledge through different categories such as 🔬 Science, ⚽ Sports, 📜 History, and 🇧🇩 Bangladesh. Players can also use special lifelines like the ✂️ 50-50 option to increase their chances of answering correctly.

# 🚀 How to Run

## ⚙️ Compile

```bash
gcc main.c -o hotseat
```

## ▶️ Run

```bash
./hotseat
```

For Windows:

```bash
hotseat.exe
```

---

# 🎯 Gameplay

1. ▶️ Start the game
2. 📂 Choose a category
3. ❓ Answer MCQ questions
4. ✂️ Use the 50-50 lifeline once during the game
5. 🏅 Earn points for correct answers
6. 👑 Try to achieve the highest score

---

# 🔥 Sample Features

## ✂️ 50-50 Lifeline

The 50-50 option removes two wrong answers from the question, leaving one correct answer and one incorrect answer.
